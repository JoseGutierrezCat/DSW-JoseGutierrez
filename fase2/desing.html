Sintaxis y Selectores CSS

CSS

La sigla CSS corresponde a la expresión inglesa Cascading StyleSheets, que puede traducirse como “Hojas de estilo en cascada”.


  • Sintaxis CSS

    • Las sintaxis son un conjunto de reglas de CSS las cuales se encuentran confomadas por dos componentes, entre los cuales encontramos un selector y un bloque de declaración:

    Ejemplo

    Selector: h1

    Declaración: {color:blue; font-size:12px;}

    En el ejemplo anterior se puede apreciar como se aplica el selector h1 para la aplicación de los elementos, mientras que la declaración la encontramos dentro de las llaves y en esta podemos guardar varias definiciones.

  • Selectores CSS

    • Cuando nos referimos a los selectores encontramos que estos se dividen en varias clases dentro de los cuales encontramos:

    1. Selectores de Grupo

    Es utilizado cuando se requiere emplear los mismos estilos a un grupo determinado de elementos

    Ejemplo

    h1,h2,h3,h4,h5,h6
    {color:green}

    2. Selectores de Clase

    Los selectores de clase nos permiten establecer estilos a un mismo elemento con atributos de la clase en especifico.

    Ejemplo

    .center
    {text-aling:center; color:red};

    3. Selectores de Identificación

    El selector de ID emplea el atributo ID de un elemento de HTML para seleccionar un elemento en especifico.

    Ejemplo

    id="para1"
    {text-aling:center; color:red};
